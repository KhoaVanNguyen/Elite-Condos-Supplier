struct StarLayer

Creates a layer with a single star in it.

  • create(_:size:lineWidth:fillColor:strokeColor:)

    Creates a square layer with given size and draws the star shape in it.

    Declaration

    Swift

    static func create(_ starPoints: [CGPoint], size: Double,
    lineWidth: Double, fillColor: UIColor, strokeColor: UIColor) -> CALayer

    Parameters

    starPoints

    Array of points for drawing a closed shape. The size of enclosing rectangle is 100 by 100.
    size

    The width and height of the layer. The star shape is scaled to fill the size of the layer.
    lineWidth

    The width of the star stroke.
    fillColor

    Star shape fill color. Fill color is invisible if it is a clear color.
    strokeColor

    Star shape stroke color. Stroke is invisible if it is a clear color.

    Return Value

    New layer containing the star shape.

  • createStarPath(_:size:lineWidth:)

    Creates a path for the given star points and size. The star points specify a shape of size 100 by 100. The star shape will be scaled if the size parameter is not 100. For exampe, if size parameter is 200 the shape will be scaled by 2.

  • scaleStar(_:factor:lineWidth:)

    Scale the star points by the given factor.

    Declaration

    Swift

    static func scaleStar(_ starPoints: [CGPoint], factor: Double, lineWidth: Double) -> [CGPoint]

    Parameters

    starPoints

    Array of points for drawing a closed shape. The size of enclosing rectangle is 100 by 100.

    factor

    The factor by which the star points are scaled. For example, if it is 0.5 the output points will define the shape twice as small as the original.

    Return Value

    The scaled shape.

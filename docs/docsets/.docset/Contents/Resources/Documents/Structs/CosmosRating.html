struct CosmosRating

Helper functions for calculating rating.

  • starFillLevel(ratingRemainder:fillMode:)

    Returns a decimal number between 0 and 1 describing the star fill level.

    Declaration

    Swift

    static func starFillLevel(ratingRemainder: Double, fillMode: StarFillMode) -> Double

    Parameters

    ratingRemainder

    This value is passed from the loop that creates star layers. The value starts with the rating value and decremented by 1 when each star is created. For example, suppose we want to display rating of 3.5. When the first star is created the ratingRemainder parameter will be 3.5. For the second star it will be 2.5. Third: 1.5. Fourth: 0.5. Fifth: -0.5.
    fillMode

    Describe how stars should be filled: full, half or precise.

    Return Value

    Decimal value between 0 and 1 describing the star fill level. 1 is a fully filled star. 0 is an empty star. 0.5 is a half-star.

  • roundFillLevel(_:fillMode:)

    Rounds a single star’s fill level according to the fill mode. Full mode returns 0 or 1 by using the standard decimal rounding. Half mode returns 0, 0.5 or 1 by rounding the decimal to closest of 3 values. Precise mode will return the fill level unchanged.

    Declaration

    Swift

    static func roundFillLevel(_ starFillLevel: Double, fillMode: StarFillMode) -> Double

    Parameters

    starFillLevel

    Decimal number between 0 and 1 describing the star fill level.
    fillMode

    Fill mode that is used to round the fill level value.

    Return Value

    The rounded fill level.

  • displayedRatingFromPreciseRating(_:fillMode:totalStars:)

    Helper function for calculating the rating that is displayed to the user taking into account the star fill mode. For example, if the fill mode is .half and precise rating is 4.6, the displayed rating will be 4.5. And if the fill mode is .full the displayed rating will be 5.

    Declaration

    Swift

    static func displayedRatingFromPreciseRating(_ preciseRating: Double,
    fillMode: StarFillMode, totalStars: Int) -> Double

    Parameters

    preciseRating

    Precise rating value, like 4.8237
    fillMode

    Describe how stars should be filled: full, half or precise.
    totalStars

    Total number of stars.

    Return Value

    Returns rating that is displayed to the user taking into account the star fill mode.

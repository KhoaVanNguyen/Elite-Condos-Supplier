  • accessibilityValue(_:rating:settings:)

    Returns the rating that is used as accessibility value. The accessibility value depends on the star fill mode.

    For example, if rating is 4.6 and fill mode is .half the value will be 4.5. And if the fill mode if .full the value will be 5.

    Declaration

    Swift

  • accessibilityIncrement(_:settings:)

    Returns the amount of increment for the rating. When .half and .precise fill modes are used the rating is incremented by 0.5.
